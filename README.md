# MTSL learner

```bash
cargo run {dictionary} {model-name}
```

Put your dictionary file in the `dictionaries` directory. The dictionary file should be in the [MFA format](https://montreal-forced-aligner.readthedocs.io/en/latest/user_guide/dictionary.html).

Supported models:

- `tsl`
- `mtsl`
- `mtsl-liberal`
